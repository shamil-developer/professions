# Консалтинг (управленческий, IT, технический)

- **Консультант по управлению (Management Consultant)** — решает бизнес‑задачи клиентов и готовит рекомендации.
- **Старший консультант** — ведёт треки и менторит команду проекта.
- **Руководитель проектов (консалтинг)** — управляет проектом и взаимоотношениями с клиентом.
- **Менеджер/партнер консалтинговой практики** — развивает практику и продажи.
- **Консультант по стратегии** — формирует стратегические инициативы и портфели.
- **Консультант по операционной эффективности** — оптимизирует процессы и издержки.
- **Консультант по организационному дизайну** — строит структуры и роли организаций.
- **Консультант по цифровой трансформации** — внедряет цифровые решения и таргет‑операмодель.
- **ИТ‑консультант** — проектирует ИТ‑решения и roadmaps.
- **Технический консультант** — решает системно‑технические задачи внедрений.
- **Консультант ERP/CRM/BI** — внедряет корпоративные платформы и аналитику.
- **Консультант по бизнес‑процессам (BPM)** — моделирует и улучшает процессы.
 - **Консультант по управлению изменениями (Change Management)** — готовит организацию к изменениям.
 - **Консультант по рискам/внутреннему контролю** — строит систему рисков и контролей.
 - **Консультант по устойчивому развитию/ESG** — помогает внедрять ESG‑повестку.
 - **Консультант по данным и ИИ (Data/AI)** — выявляет кейсы и внедряет аналитические решения.
 - **Аналитик консалтинга** — собирает данные и проводит анализ.
 - **Ассоциированный партнер** — развивает ключевых клиентов и практики.
