# Качество, стандартизация и сертификация (QA/QC)

- **Инженер по качеству (QA)** — выстраивает процессы и методы обеспечения качества.
- **Инженер по контролю качества (QC)** — проверяет продукцию и процессы на соответствие.
- **Контролер ОТК** — проводит операционный и приемочный контроль.
- **Инженер по стандартизации** — разрабатывает и поддерживает стандарты предприятия.
- **Менеджер системы менеджмента качества (QMS/ISO 9001)** — поддерживает СМК и аудиты.
- **Внутренний аудитор (ISO)** — проверяет соответствие требованиям стандартов.
- **Инженер по метрологии** — обеспечивает поверку и калибровку средств измерений.
- **Специалист по сертификации продукции** — организует процедуры сертификации.
- **Инженер по валидации (GxP)** — валидирует процессы и оборудование в регулируемых отраслях.
- **Инспектор испытательной лаборатории** — проводит испытания и оформляет протоколы.
 - **Инженер по качеству поставщиков (SQE)** — развивает качество у поставщиков, APQP/PPAP.
 - **Менеджер по качеству клиентов (Customer Quality)** — работа с рекламациями и удовлетворённостью.
 - **Специалист по статистическому управлению процессами (SPC)** — внедряет контроль карт и капабилити.
 - **Координатор CAPA** — управляет корректирующими и предупреждающими действиями.
 - **Менеджер по аудиту поставщиков** — оценивает поставщиков и планы улучшений.
 - **Руководитель отдела качества** — управляет командой и стратегией качества.
