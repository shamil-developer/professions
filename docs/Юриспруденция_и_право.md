# Юриспруденция и право

- **Адвокат** — представляет интересы клиентов в суде и переговорах.
- **Юрисконсульт** — обеспечивает правовую поддержку компании.
- **Корпоративный юрист** — ведёт корпоративные процедуры и сделки.
- **Юрист по договорной работе** — готовит и согласует договоры.
- **Налоговый юрист** — консультирует по налоговым рискам и спорам.
- **Трудовой юрист** — решает трудовые споры и вопросы HR‑права.
- **Юрист по интеллектуальной собственности (IP)** — регистрирует и защищает IP‑права.
- **Патентный поверенный** — патентует изобретения и защищает права.
- **Юрист по недвижимости** — сопровождает сделки с объектами.
- **Юрист по банкротству** — ведёт процедуры несостоятельности.
- **Судебный юрист (литигатор)** — ведёт судебные процессы.
- **Медиатор/арбитр** — урегулирует споры вне суда.
- **Нотариус** — удостоверяет сделки и документы.
- **Юрист по антимонопольному праву (антитраст)** — консультирует по конкуренции и проверкам.
- **Юрист по международному праву** — сопровождает трансграничные вопросы.
- **Юрист по уголовному праву** — защищает/представляет в уголовных делах.
- **Юрист‑комплаенс (Compliance)** — обеспечивает соблюдение норм и санкционных режимов.
- **Data Protection Officer (DPO)** — отвечает за защиту персональных данных.
- **Помощник адвоката/юриста** — готовит документы и исследования.
- **Помощник судьи** — обеспечивает процессуальную подготовку дел.
 - **Прокурор** — поддерживает обвинение в суде.
 - **Судья** — рассматривает дела и выносит решения.
