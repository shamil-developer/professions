# Авиация и космос

- **Командир воздушного судна (КВС)** — командует полётом и экипажем.
- **Второй пилот (FO)** — помогает КВС в пилотировании и навигации.
- **Бортпроводник (кабинный экипаж)** — обеспечивает безопасность и сервис на борту.
- **Диспетчер УВД (ATC)** — управляет воздушным движением.
- **Диспетчер по планированию полетов (OCC)** — планирует рейсы и сопровождение.
- **Специалист по безопасности полетов (Flight Safety)** — предотвращает инциденты и анализирует риски.
- **Агент наземного обслуживания (Ground Handling)** — оформляет рейсы и пассажиров/багаж.
- **Рампа‑агент/авиагрузчик** — обслуживает борт на перроне и багажные операции.
- **Специалист по опасным грузам (авиа)** — контролирует перевозку DG по правилам ICAO/IATA.
- **Авиационный техник (ТО)** — выполняет техническое обслуживание ВС.
- **Инженер по ПиД (планер и двигатель)** — отвечает за планер/двигатели и ресурсы.
- **Инженер по авионике** — обслуживает электронику и системы управления.
- **Техник по авионике** — проводит работы по системам бортовой электроники.
- **Инженер по авиационным двигателям** — анализирует состояние и ремонт ДС.
- **Специалист CAMO/Continuing Airworthiness** — ведёт летную годность парка.
- **Планировщик ТО (Maintenance Planner)** — формирует план ТО и ресурсы.
- **Конструктор авиастроения** — проектирует конструкции ЛА.
- **Инженер‑аэродинамик** — рассчитывает аэродинамику и испытания.
- **Инженер ракетно‑космической техники** — проектирует РКТ и узлы.
- **Оператор космических аппаратов (ЦУП)** — управляет спутниками и телеметрией.
 - **Инженер по телеметрии/наземным комплексам** — поддерживает наземные станции.
 - **Инженер по спутниковым системам/ГНСС** — развивает навигационные системы и сервисы.
