# Недвижимость и девелопмент (включая управление недвижимостью)

- **Девелопер проектов (Project Developer)** — формирует концепции и реализует проекты от земли до ввода.
- **Менеджер по девелопменту** — координирует стадии проектирования, согласований и строительства.
- **Аналитик девелопмента** — считает экономику, рынок и сценарии проектов.
- **Менеджер проекта девелопмента** — управляет сроками, бюджетом и рисками проекта.
- **Координатор ввода в эксплуатацию** — организует получение ЗОС и ввод объекта.
- **Агент по недвижимости (риелтор)** — сопровождает сделки купли‑продажи/аренды.
- **Брокер по коммерческой недвижимости** — ведёт сделки по офисам, ретейлу, складам.
- **Менеджер по продажам недвижимости** — продаёт объекты/квартиры застройщика.
- **Менеджер по аренде** — ищет арендаторов и ведёт договоры.
- **Менеджер по работе с арендаторами (Tenant Relations)** — решает операционные вопросы арендаторов.
- **Оценщик недвижимости** — оценивает рыночную стоимость объектов.
- **Инвестиционный аналитик (Real Estate)** — анализирует инвестиции и структуру сделок.
- **Property Manager (управляющий объектом)** — управляет эксплуатацией, сервисами и арендаторами.
- **Facility Manager (эксплуатация объектов)** — обеспечивает инженерную эксплуатацию и подрядчиков.
- **Главный инженер по эксплуатации (Chief Engineer FM)** — руководит инженерной службой объекта.
- **Инженер по эксплуатации зданий** — обслуживает инженерные системы здания.
- **Техник по эксплуатации зданий** — выполняет регламентные работы и мелкий ремонт.
- **Юрист по недвижимости** — ведёт сделки, права, аренду и землевопросы.
- **Кадастровый инженер** — выполняет межевание и кадастровые работы.
- **Специалист по земельным вопросам** — оформляет права на землю и сервитуты.
- **Asset Manager (портфель недвижимости)** — управляет доходностью и стоимостью активов.
 - **Брокер инвестиционных сделок** — структурирует и закрывает сделки купли‑продажи.
 - **Специалист по due diligence объектов** — проверяет юридические/тех. риски перед сделкой.
