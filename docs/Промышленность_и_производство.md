# Промышленность и производство

- **Инженер‑технолог** — разрабатывает технологии, карты и режимы изготовления продукции.
- **Мастер смены** — руководит сменой, распределяет задачи и контролирует выполнение.
- **Начальник цеха** — отвечает за производственный участок, ресурсы и показатели.
- **Директор по производству** — управляет всей производственной деятельностью предприятия.
- **Инженер по производственному планированию (PPC)** — планирует объемы, мощности и загрузку.
- **Планировщик производства** — формирует графики и очередность операций.
- **Инженер по качеству (QA/QC)** — обеспечивает соответствие продукции требованиям.
- **Контролер ОТК** — проводит входной/операционный/приемочный контроль.
- **Инженер по промышленной безопасности** — снижает риски аварий и контролирует ОПО.
- **Инженер по охране труда** — внедряет систему ОТ и обучение персонала.
- **Инженер‑конструктор (CAD)** — проектирует изделия и КД в CAD‑системах.
- **Инженер по автоматизации (АСУ ТП)** — внедряет и обслуживает системы управления.
- **Инженер‑робототехник** — интегрирует промышленные роботы и манипуляторы.
- **Инженер по техпроцессам (Process Engineer)** — оптимизирует процессы и оборудование.
- **Технолог‑металлург** — разрабатывает режимы литья/прокатки/термообработки.
- **Технолог‑химик** — ведёт химпроцессы и рецептуры.
- **Технолог деревообработки** — задаёт режимы обработки древесины.
- **Оператор станков с ЧПУ** — обрабатывает детали по программам.
- **Наладчик ЧПУ** — настраивает станки и оснастку под партии.
- **Токарь** — выполняет токарные операции.
- **Фрезеровщик** — выполняет фрезерные операции.
- **Сварщик (производство)** — сваривает конструкции по технологии.
- **Слесарь‑ремонтник** — ремонтирует оборудование и узлы.
- **Электромонтер по ремонту** — обслуживает электрооборудование.
- **Механик** — поддерживает исправность механического парка.
- **Оператор производственной линии** — ведёт технологический процесс на линии.
- **Инженер по логистике производства** — обеспечивает поставки на линии (intralogistics).
- **Инженер по упаковке** — проектирует упаковку и процессы упаковывания.
 - **Специалист по бережливому производству (Lean/Kaizen)** — устраняет потери и улучшает потоки.
 - **Специалист по 5S/TPM** — внедряет стандарты порядка и ТО оборудования.
