# Государственная служба и публичное управление

- **Государственный гражданский служащий (специалист)** — исполняет функции органа власти по направлению.
- **Муниципальный служащий (специалист)** — решает задачи местного самоуправления.
- **Начальник отдела (госорган)** — руководит подразделением и проектами.
- **Руководитель департамента/управления** — формирует политику и KPI направления.
- **Советник/помощник руководителя** — готовит материалы и координирует поручения.
- **Аналитик государственной политики (Public Policy)** — оценивает меры политики и эффекты.
- **Эксперт по ГЧП/проектам развития** — готовит и сопровождает проекты ГЧП.
- **Специалист по государственным программам** — ведёт программы и отчётность.
- **Государственный инспектор (надзор/контроль)** — осуществляет проверки и предписания.
- **Специалист по нормативно‑правовой работе** — разрабатывает НПА и правовые заключения.
- **Специалист по международным связям (в госорганах)** — координирует международные контакты.
- **Специалист по госзакупкам/контрактный управляющий** — проводит закупки по 44‑ФЗ/223‑ФЗ.
- **Пресс‑секретарь госоргана** — строит коммуникации с общественностью.
- **Руководитель проектного офиса (в госуправлении)** — внедряет проектные практики в органе власти.
- **Менеджер проекта (в госорганах)** — реализует проекты и контролирует сроки/результаты.
- **Аналитик данных (в госуправлении)** — анализирует данные для решений и сервисов.
 - **Специалист по делопроизводству (госслужба)** — ведёт официальную документацию и архив.
 - **Специалист по кадровой работе (госслужба)** — кадровое делопроизводство и конкурсные процедуры.
