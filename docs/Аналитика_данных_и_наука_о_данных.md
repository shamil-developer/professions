# Аналитика данных и наука о данных

- **Аналитик данных (Data Analyst)** — собирает, очищает и анализирует данные для решений.
- **Продуктовый аналитик** — оценивает метрики продукта и воронки.
- **BI‑аналитик** — строит отчёты и витрины данных.
- **Аналитик маркетинга** — анализирует кампании и каналы привлечения.
- **Аналитик продаж (Sales Analyst)** — анализирует продажи, план/факт и прогнозы.
- **Финансовый аналитик данных (FP&A/fin data)** — поддерживает модели и отчёты финансистов.
- **Аналитик CRM/Retention** — сегментация, удержание и LTV.
- **Аналитик экспериментов (A/B)** — проектирует и анализирует эксперименты.
- **Аналитик по ценообразованию (Pricing Analyst)** — моделирует цены и эластичность.
- **Аналитик по качеству данных (Data Quality)** — контролирует качество и справочники.
- **Data Scientist** — строит модели ML и аналитику.
- **Research Data Scientist** — исследует новые методы и подходы ML.
- **Машинный обученец (ML Engineer)** — внедряет и масштабирует ML‑модели.
- **MLOps‑инженер** — автоматизирует CI/CD для ML и мониторинг.
- **Инженер данных (ETL/ELT)** — строит пайплайны и DWH/даталейки.
- **Архитектор данных** — проектирует архитектуру данных и интеграции.
- **Аналитик Big Data** — анализирует большие объёмы и стриминг.
- **Инженер Spark/Hadoop** — разрабатывает вычислительные пайплайны на платформах.
- **Визуализатор данных (Data Viz)** — создаёт дашборды и визуализации.
- **Разработчик BI (Power BI/Tableau/Qlik)** — разрабатывает BI‑модули и модели данных.
- **Специалист по мастер‑данным (MDM)** — управляет эталонными данными.
- **Data Steward (управление данными)** — отвечает за правила и качество доменов данных.
- **Аналитик геоданных (GIS Analyst)** — анализирует пространственные данные.
- **Специалист NLP** — разрабатывает решения обработки текста.
- **Специалист Computer Vision** — строит модели обработки изображений/видео.
- **Специалист по рекомендательным системам** — проектирует рекомендатели и ранжирование.
- **Эконометрист** — строит эконометрические модели и прогнозы.
 - **Статистик** — разрабатывает методы и проверяет гипотезы.
 - **Аналитик риск‑моделей** — разрабатывает и валидацирует модели рисков.
