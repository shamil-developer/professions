# Фармация и биотехнологии

- **Провизор** — консультирует по лекарствам, контролирует отпуск и качество.
- **Фармацевт (аптечный специалист)** — отпускает ЛС и ведёт учёт в аптеке.
- **Заведующий аптекой** — управляет аптекой, персоналом и ассортиментом.
- **Клинический фармаколог** — оптимизирует лекарственную терапию пациентов.
- **Фармацевт‑технолог** — разрабатывает и производит лекарственные формы.
- **Специалист по регистрации лекарств (Regulatory Affairs)** — готовит досье и взаимодействует с регулятором.
- **Специалист по фармаконадзору (PV)** — мониторит безопасность и побочные реакции.
- **Медицинский представитель (фарма)** — продвигает препараты среди врачей.
- **Региональный менеджер (Pharma KAM)** — развивает ключевых клиентов и продажи.
- **CRA (координатор клинических исследований)** — мониторит центры и соблюдение протокола.
- **CRA Manager** — руководит командой мониторинга и проектами.
- **CRC (координатор центра)** — ведёт документацию и процесс в клиническом центре.
- **Data Manager (клинические исследования)** — управляет базами данных исследований.
- **Биостатистик** — проектирует статистику и анализ данных исследований.
- **Биотехнолог** — разрабатывает биопроцессы и культивирование.
- **Специалист по разработке лекарственных форм** — выбирает составы и технологии производства.
- **Лаборант биотехнологического производства** — выполняет операции и анализы на производстве.
- **Специалист по качеству (QA/QC, GMP)** — контролирует соответствие GMP и качество.
- **Специалист по валидации процессов** — валидирует процессы и оборудование.
 - **Инженер по оборудованию фармпроизводства** — обслуживает и внедряет фарм‑оборудование.
 - **Специалист по холодовой цепи (Cold Chain)** — обеспечивает температурный режим логистики.
