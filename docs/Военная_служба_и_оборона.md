# Военная служба и оборона

- **Военнослужащий (общевойсковые специальности)** — выполняет задачи по службе в своем роде войск.
- **Офицер (командир взвода/роты/батальона)** — руководит подразделением и боевой подготовкой.
- **Офицер штаба (оперативное планирование)** — планирует операции и обеспечивает управление.
- **Военный инженер** — решает инженерно‑технические задачи войск.
- **Связист (военная связь)** — обеспечивает устойчивую связь подразделений.
- **Специалист ПВО** — обслуживает средства противовоздушной обороны.
- **Специалист РЭБ (радиоэлектронная борьба)** — подавляет и защищает радиосредства.
- **Танкист** — управляет танком и вооружением.
- **Артиллерист** — ведёт огонь артиллерийских систем.
- **Пехотинец (мотострелковые войска)** — выполняет задачи на земле в составе подразделения.
- **Морской пехотинец** — ведёт боевые действия на море и суше.
- **Летчик (военная авиация)** — выполняет боевые вылеты и задачи.
- **Штурман (военная авиация)** — отвечает за навигацию и цели.
- **Техник по вооружению** — обслуживает и ремонтирует вооружение.
- **Инженер по эксплуатации авиационной техники** — поддерживает исправность авиатехники.
- **Инженер ракетно‑артиллерийского вооружения** — обслуживает ракетно‑артиллерийские системы.
- **Сапер (инженерные войска)** — проводит инженерные работы, разминирование.
- **Разведчик** — добывает сведения о противнике.
- **Снайпер** — поражает цели на дальних дистанциях.
- **Инструктор боевой подготовки** — обучает личный состав.
 - **Военный медик** — оказывает медицинскую помощь в войсках.
 - **Контрактник/служащий по контракту** — проходит службу по контракту на должности.
