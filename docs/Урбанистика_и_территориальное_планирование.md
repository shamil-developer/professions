# Урбанистика и территориальное планирование

- **Урбанист** — анализирует городскую среду и формирует сценарии развития.
- **Градостроитель** — разрабатывает генпланы, ПЗЗ и правила землепользования.
- **Планировщик территории** — планирует функциональное зонирование и плотности.
- **Архитектор‑градостроитель** — проектирует кварталы и общественные пространства.
- **Транспортный планировщик** — моделирует трафик и схемы движения.
- **Инженер улично‑дорожной сети** — проектирует улицы, развязки и сечения.
- **Специалист по общественным пространствам (placemaking)** — развивает парки и площади с участием жителей.
- **Аналитик городских данных** — обрабатывает мобилити/сервисные данные для решений.
- **Специалист ГИС (городские данные)** — создает карты и геосервисы для планирования.
- **Специалист по мастер‑планам** — готовит стратегические документы развития территорий.
- **Специалист по нормотворчеству (градостроительное право)** — разрабатывает регламенты и НПА.
- **Координатор общественных обсуждений/слушаний** — организует диалог с жителями по проектам.
 - **Эколог градостроительных проектов** — оценивает экологические эффекты планировок.
 - **Менеджер по развитию районов** — инициирует и сопровождает проекты локального развития.
