# Экология, охрана природы и устойчивое развитие

- **Эколог** — оценивает воздействие на среду и разрабатывает природоохранные меры.
- **Инженер‑эколог** — внедряет экологические решения на предприятиях и ведёт отчётность.
- **Специалист по ОВОС/ОВНС (оценка воздействия)** — проводит оценку воздействия, общественные обсуждения и готовит отчёты.
- **Специалист по экологическому менеджменту (ISO 14001)** — строит и поддерживает СЭМ, проводит внутренние аудиты.
- **Специалист по управлению отходами** — организует обращение с отходами: паспортизацию, хранение, утилизацию.
- **Специалист по рекультивации земель** — проектирует и контролирует восстановление нарушенных территорий.
- **Экотоксиколог** — оценивает токсичность веществ и риски для экосистем и человека.
- **Специалист по биоразнообразию** — мониторит виды и разрабатывает планы их сохранения.
- **Специалист по водным ресурсам (гидроэколог)** — управляет качеством, использованием и охраной вод.
- **Специалист по климатической политике/адаптации** — анализирует климатические риски и меры адаптации.
- **Специалист по углеродному учету/CFootprint** — рассчитывает выбросы GHG и управляет углеродным следом.
- **ESG‑специалист** — внедряет ESG‑практики и готовит нефинансовую отчётность.
- **Аналитик устойчивого развития** — собирает метрики и анализирует прогресс по целям устойчивости.
 - **Специалист по LCA (жизненный цикл продукта)** — проводит анализ жизненного цикла и экологического следа.
 - **Менеджер циклической экономики (Circular Economy)** — внедряет повторное использование и снижение отходов.
 - **Эколог‑аудитор** — проводит независимые экологические аудиты и проверки.
 - **EHS‑менеджер (экология, охрана труда, безопасность)** — интегрирует требования E, H, S на предприятии.
 - **Специалист по качеству атмосферного воздуха/выбросам** — мониторинг, расчёты и НДВ.
 - **Специалист по шуму и вибрации** — измеряет и снижает акустическое воздействие.
 - **Консультант по устойчивому финансированию (Sustainable Finance)** — структурирует зелёные инструменты и критерии таксономий.
 - **Природоохранный инспектор** — осуществляет надзор за соблюдением природоохранных требований.
